ℹ️ How Temperature Calibration Works

This system automatically learns the temperature offset for each zone by comparing your AC unit's setpoint with an external temperature sensor. Once learned, it automatically adjusts the AC setpoint so your external sensor reaches the target temperature.

1
Set Target Temperature
You set the desired temperature (e.g., 22°C) on the AC unit or dashboard
2
System Observes Difference
The system compares the setpoint with your external sensor reading
3
Learning Phase
Over 10+ samples, the system calculates the average offset for that zone
4
Auto Adjustment
Once learned, the system automatically adjusts AC setpoint to reach your target temp
5
Lock Calibration
Lock the calibration to prevent further learning and keep settings consistent
💡 Tip: The system is designed to stay within ±2°C tolerance. If a zone is consistently off by more than 2°C, learning will continue until it reaches the target. Once locked, manual adjustments won't trigger learning.

💡 Tips for Best Calibration

  • Run AC continuously: Let AC run for 30+ minutes to establish offset pattern
  • Avoid rapid changes: Don't change setpoint too frequently during learning
  • Place sensor correctly: Ensure external sensor is in the zone (not in AC outlet)
  • Check sensor accuracy: Compare external sensor with other thermometers
  • Lock when stable: After calibration is consistent, lock to prevent drift
  • Seasonal check: Recalibrate seasonally as AC efficiency changes
